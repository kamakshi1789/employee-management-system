# Employee Management System (Python)

A console-based Employee Management System developed using Python.
This project uses file handling to manage employee records and salary data.

## Features
- Admin Login & Employee Login
- Add, Modify, Delete, Search Employees
- Auto Employee ID generation
- Salary Management & History
- Employee Dashboard
- Data Backup
- Console-based Menu System

## Technologies Used
- Python
- File Handling
- OS & MSVCRT modules
